Здесь всего 2 программы: головная Index.f90  и   MODULES.f90.
Для получении исполняемого файла  gfortran Index.f90  MODULES.f90 –o GO
С первого раза GO не получается (что-то с модулем). Вводи эту строчку 2-й раз.
Файл для управления  UV-CONTROL_FILE   
Пример:  
/Input/day
./Input/ozone
./Input/orogr
./Input/aeros
./Input/albed
./Input/cloud
Calc_12Dec
Входные файлы в ./Input/ с необходимой информацией о дате (день в году), озоне, высоте местности, аэрозоле и облачности (посмотри их, если надо поясню).
В главный для графика файл в ./Outfile/  В примере это будет Calc_12Dec
